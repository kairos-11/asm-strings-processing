# Обработка строк

## Описание задачи
Программа считывает текстовые строки из стандартного потока ввода. Для каждой строки выполняется фильтрация: удаляются все слова, первый символ которых не совпадает с первым символом первого слова текущей строки. Результат обработки записывается в файл, имя которого запрашивается у пользователя в диалоговом режиме.

## Условия и ограничения
- Ввод данных: стандартный поток ввода (`stdin`). Завершение ввода осуществляется комбинацией `Ctrl+D`.
- Вывод данных: файл. Имя файла передаётся через интерактивный запрос во время выполнения.
- Форматирование: слова в результирующих строках разделяются одиночными пробелами. Наличие пробелов в начале и конце строки не допускается.
- Количество выходных строк строго соответствует количеству входных. Пустые строки сохраняются.
- Поддержка обработки строк произвольной длины.
- Запрещено использование внешних библиотек и системного вызова `mmap()`.
- Реализована корректная обработка ошибок ввода/вывода и выделения памяти.

## Технические параметры
- Архитектура: x86_64
- Операционная система: Linux
- Язык: Ассемблер (NASM)
- Взаимодействие с ОС: прямые системные вызовы (`syscall`), динамическая память через `sys_brk`.

## Сборка и запуск
Проект компонуется с использованием `nasm` (формат `elf64`) и `ld` в `Makefile`. После запуска программа запрашивает имя выходного файла, после чего ожидает ввод данных. Для проверки корректности форматирования и отсутствия скрытых символов рекомендуется использовать побайтовый просмотр результирующего файла.
